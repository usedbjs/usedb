Installation

PreRequisite#

useDB has a peer dependency on mobx-react.

  • NPM
  • Yarn
npm install mobx-react
yarn add mobx-react

Installing UseDB#

  • NPM
  • Yarn
npm install @usedb/core @usedb/react
yarn add @usedb/core @usedb/react
